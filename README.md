# Alpha Business Academy — O Caminho do Dono

Landing page de alta conversão para a imersão presencial **"O Caminho do Dono"**.

## 🚀 Tecnologias
- HTML5 / CSS3 (Aura x Gemini Design System v3.0)
- Tailwind CSS (via CDN para rapidez de deploy)
- Lucide Icons
- Scroll Reveal Animations

## 📂 Estrutura do Projeto
- `index.html`: Página principal com foco em conversão e dor/transformação.
- `assets/css/`: Brand tokens e estilos customizados.
- `assets/img/`: Fotos dos mentores e imagens de transformação.
- `assets/videos/`: Depoimentos de alunos em vídeo.
- `COPY_STRATEGY.md`: Estratégia de copy e gatilhos mentais utilizados.

## 🛠️ Como editar
1. Para alterar a headline ou textos, edite o `index.html`.
2. As cores e fontes estão definidas nos arquivos de CSS dentro de `assets/css/`.
3. Para novas imagens, coloque-as em `assets/img/` e atualize os caminhos no HTML.

## 🎯 Objetivo
Transformar o dono de negócio de "escravo da operação" em "CEO de Elite" através dos 12 Pilares Estratégicos.
